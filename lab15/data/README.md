Overview

This repository contains data, code, and supplemental materials associated with our study on mammalian trait evolution.

all_mammals.newick: Phylogenetic tree of all mammalian species in Newick format.
residuals.csv: CSV file containing residual values from statistical analyses.
Data_S1.csv: Primary data file used for analysis, including species traits and ecological variables.

Key Variables of Data_S1:
Body Mass: The body mass of each mammalian species, measured in grams (g)
Longevity: The expected lifespan of each species, measured in years.
Gestation Length: The length of pregnancy for each species, measured in days

How to Use the Files

Data files (all_mammals.newick, residuals.csv, Data_S1.csv) can be used with phylogenetic and statistical software to reproduce analyses presented in the study.
Code files (.r scripts and .zip files) contain the R scripts used to generate the figures in the manuscript. Unzip the .zip files and follow the instructions in the accompanying README inside each folder to run the scripts.
Supplementary materials provide additional context, methods, and data used in the study.
